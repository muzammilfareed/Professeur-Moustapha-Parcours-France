# Professeur Moustapha : Parcours France
 
An EGF game created by Hervé Yvis. Guided by Professor Moustapha, you explore France in four chapters: symbols of the Republic, heritage, secularism, and geographic landmarks. A quiz game and interactive activities to learn while having fun and to make integration in France easier.
 
This game is directly inspired by the game "Abdoul apprend l'alphabet français" (https://apkpure.net/abdoul-apprend-l-alphabet-fran%C3%A7ais/com.yvisherve.GAME.AlphabetFrancais) and is released under the Creative Commons Attribution 4.0 International (CC BY 4.0) license: https://creativecommons.org/licenses/by/4.0/deed.
